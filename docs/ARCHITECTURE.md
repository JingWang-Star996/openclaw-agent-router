## 🎯 系统概述

OpenClaw Agent Router 是一个智能 Agent 路由系统，核心目标是：
1. **自动调用** - 根据任务类型自动调用最合适的 Agent
2. **智能推荐** - 不满足自动调用条件时，主动推荐相关 Agent
3. **实时扫描** - 实时扫描 `agents/` 文件夹，动态获取可用 Agent 列表

## 📦 核心模块

### 1. Agent Scanner（扫描器）

**职责**: 实时扫描 `agents/` 文件夹，解析 Agent 信息

**输入**: 
- `agents/` 文件夹路径

**输出**:
```javascript
[
  {
    name: 'AI 数据分析师',
    description: '专业分析游戏数据...',
    capabilities: ['数据分析', '留存分析', '付费分析'],
    keywords: ['数据', '留存', '付费', '分析'],
    category: '运营'
  },
  // ...
]
```

**缓存策略**:
- 同一次对话缓存 5 分钟
- 避免重复扫描

---

### 2. Router（路由器）

**职责**: 路由决策核心

**流程**:
```
任务 → 判断自动调用条件 → 是 → 调用 Agent
                      ↓ 否
                      评估复杂度
                      ↓
            简单 → 自己处理
            中等 → 推荐 1-3 个 Agent
            复杂 → 推荐 3-5 个 Agent（协作）
```

**自动调用规则**:
| 任务类型 | 关键词 | 调用 Agent |
|----------|--------|-----------|
| 游戏设计 | 设计/玩法/系统/数值 | AI 策划团队 |
| 数据分析 | 留存/付费/CVR/ 数据 | AI 数据分析师 |
| 项目管理 | 排期/进度/资源 | PMO 项目经理 |
| 文章写作 | 文章/文档/周报 | 编辑部 |
| 代码技术 | 代码/Bug/ 架构 | AI 程序员 |

---

### 3. Matcher（匹配器）

**职责**: 多维度匹配任务和 Agent

**匹配维度**:
1. **关键词匹配** (10 分/词)
   - 任务描述包含 Agent 关键词
   
2. **描述匹配** (20 分)
   - Agent 职责描述包含任务关键词
   
3. **分类匹配** (15 分)
   - 任务类型匹配 Agent 分类
   
4. **能力匹配** (5 分/能力)
   - 任务匹配 Agent 核心能力

**评分示例**:
```
任务："分析下次留为什么只有 30%"

AI 数据分析师:
- 关键词匹配：数据 (+10), 分析 (+10), 留存 (+10) = 30 分
- 描述匹配：无 = 0 分
- 分类匹配：运营 (+15) = 15 分
- 能力匹配：数据分析 (+5) = 5 分
总计：50 分
```

---

### 4. Recommender（推荐器）

**职责**: 生成用户友好的推荐文本

**输出格式**:
```
这个任务我可以自己处理，但以下 Agent 可能更专业：

1. **AI 数据分析师**（运营）
   专业分析游戏数据，包括留存/付费/用户行为等
   匹配度：50 分

2. **AI 数值策划**（策划）
   负责数值建模、平衡性设计、付费数值设计
   匹配度：35 分

3. **AI 系统策划**（策划）
   负责系统设计、需求分析、文档撰写
   匹配度：25 分

需要我调用哪个？
```

```
│                  │ → 自动调用/推荐/自己处理

    ↓

```

## 🔒 错误处理

### 1. agents/ 文件夹不存在

```javascript
if (!fs.existsSync(agentsDir)) {
  console.warn('agents/ 文件夹不存在');
  return [];
}
```

### 2. agent.md 读取失败

```javascript
try {
  const content = fs.readFileSync(agentPath, 'utf-8');
} catch (e) {
  console.warn(`无法读取 ${agentName} 的 agent.md`);
  return null;
}
```

### 3. 扫描超时

```javascript
const timeout = setTimeout(() => {
  console.warn('Agent 扫描超时，使用缓存');
}, 5000);
```

## 🚀 扩展性

### 新增 Agent 类型

无需修改代码，只需：
1. 在 `agents/` 下创建新文件夹
2. 创建 `agent.md` 文件
3. 系统自动识别

### 新增路由规则

```javascript
const newRules = [
  {
    name: '新类型',
    keywords: ['关键词 1', '关键词 2'],
    category: '分类'
  }
];

const router = new AgentRouter({ autoCallRules: newRules });
```

### 自定义匹配算法

```javascript
function customMatch(task, agents) {
  // 自定义匹配逻辑
  return matchedAgents;
}
```
