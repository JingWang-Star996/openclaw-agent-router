## 核心功能

### 1. 自动路由

根据任务类型自动调用最合适的 AI：

| 任务类型 | 自动调用 |
|----------|----------|
| 游戏设计/数值/系统 | AI 主策划 → 分发到专业策划 |
| 留存/付费/数据分析 | AI 数据分析师 + AI 数值策划 |
| 项目管理/排期/资源 | PMO 项目经理 |
| 文章写作/周报 | 编辑部协作系统/周报 AI |
| 代码/技术 | AI 主程 |
| 美术/UX | AI 主美 + UX 设计师 |
| 人力资源 | 人力资源部门 AI |
| 质量管理 | 质量部门 AI |
| 运营活动 | 运营总监 + 用户运营 |

**判断逻辑**：
```javascript
function shouldAutoCall(task) {
  const taskLower = task.toLowerCase();
  
  for (const rule of autoCallRules) {
    for (const keyword of rule.keywords) {
      if (taskLower.includes(keyword.toLowerCase())) {
        return true;
      }
    }
  }
  
  return false;
}
```

**推荐逻辑**：
1. 评估任务复杂度
2. 实时扫描 agents/ 文件夹
3. 解析每个 AI 的 agent.md
4. 提取名称/职责/关键词
5. 智能匹配任务
6. 生成推荐列表

### 3. 实时扫描

实时扫描 `agents/` 文件夹，动态获取可用 AI 列表：

**扫描流程**：
```javascript
async function scanAgents() {
  const agentsDir = '/home/z3129119/.openclaw/workspace/agents';
  
  // 读取所有子目录
  const dirs = await fs.readdir(agentsDir, { withFileTypes: true });
  const agentDirs = dirs.filter(d => d.isDirectory());
  
  // 解析每个 AI
  const agents = [];
  for (const dir of agentDirs) {
    const agentInfo = await parseAgent(agentsDir, dir.name);
    if (agentInfo) {
      agents.push(agentInfo);
    }
  }
  
  return agents;
}
```

**优势**：
- ✅ 新增 AI 立即可用
- ✅ 删除 AI 自动失效
- ✅ 零维护成本

**缓存策略**：
- 同一次对话缓存 5 分钟
- 避免重复扫描

### 匹配算法

**多维度评分**：

1. **关键词匹配** (10 分/词)
   - 任务描述包含 AI 关键词
   
2. **描述匹配** (20 分)
   - AI 职责描述包含任务关键词
   
3. **分类匹配** (15 分)
   - 任务类型匹配 AI 分类
   
4. **能力匹配** (5 分/能力)
   - 任务匹配 AI 核心能力

**评分示例**：
```
任务："分析下次留为什么只有 30%"

AI 数据分析师:
- 关键词匹配：数据 (+10), 分析 (+10), 留存 (+10) = 30 分
- 描述匹配：无 = 0 分
- 分类匹配：运营 (+15) = 15 分
- 能力匹配：数据分析 (+5) = 5 分
总计：50 分
```
